Le Tiralark

En seulement 88 étapes

  • Étape 1 : Rassembler tous les éléments de l’arc : poignée, branche haute, branche basse, corde, fausse corde, repose-flèche, viseur, stabilisation.
  • Étape 2 : Nettoyer rapidement la poignée et les emboîtements.
  • Étape 3 : Identifier la branche haute.
  • Étape 4 : Identifier la branche basse.
  • Étape 5 : Tenir la poignée fermement, sans serrer.
  • Étape 6 : Insérer la branche basse dans son logement.
  • Étape 7 : Verrouiller la branche basse.
  • Étape 8 : Insérer la branche haute.
  • Étape 9 : Verrouiller la branche haute.
  • Étape 10 : Vérifier visuellement que les deux branches sont bien droites.
  • Étape 11 : Regarder l’arc comme un fusil pour vérifier l’axe général.
  • Étape 12 : Sortir la corde et repérer la grande boucle.
  • Étape 13 : Repérer la petite boucle de la corde (c'est l'autre côté ducoup).
  • Étape 14 : Placer la petite boucle sur la branche basse.
  • Étape 15 : Laisser la grande boucle libre sur la branche haute sans la mettre dans l’encoche.
  • Étape 16 : Prendre la fausse corde.
  • Étape 17 : Mettre la petite boucle de la fausse corde sur la branche basse.
  • Étape 18 : Mettre la grande boucle sur la branche haute.
  • Étape 19 : Vérifier que fausse corde et corde ne se croisent pas.
  • Étape 20 :Placer l’arc verticalement , branche basse au sol.
  • Étape 21 : Mettre un pied sur la fausse corde.
  • Étape 22 : Tirer la poignée vers le haut pour plier l’arc.
  • Étape 23 : Glisser la grande boucle de la corde dans l’encoche de la branche haute.
  • Étape 24 : Relâcher doucement la fausse corde.
  • Étape 25 : Retirer complètement la fausse corde.
  • Étape 26 : S’assurer que la corde est bien calée dans les encoches.
  • Étape 27 : Vérifier qu’elle n’est ni vrillée ni croisée.
  • Étape 28 : Contrôler le band avec une équerre.
  • Étape 29 : Ajuster le band en vrillant ou dévrillant la corde.
  • Étape 30 : Vérifier la symétrie des branches de face.
  • Étape 31 : Vérifier la symétrie des branches de profil.
  • Étape 32 : Repérer l’emplacement du repose-flèche sur la poignée.
  • Étape 33 : Nettoyer la zone où se trouvera le repose-flèche.
  • Étape 34 : Installer le repose-flèche autocollant ou vissé.
  • Étape 35 : Vérifier qu’il est droit et parallèle à l’axe.
  • Étape 36 : Installer le bouton Berger.
  • Étape 37 : Ajuster sa dureté.
  • Étape 38 : Placer un point d’encochage sur la corde.
  • Étape 39 : Le fixer avec une pince.
  • Étape 40 : Vérifier qu’il est à la bonne hauteur.
  • Étape 41 : Tester le glissement de la flèche sur le repose-flèche.
  • Étape 42 : Vérifier que l’encoche clips correctement la corde.
  • Étape 43 : Regarder le tiller.
  • Étape 44 : Ajuster le tiller si nécessaire.
  • Étape 45 : Monter le viseur sur la poignée.
  • Étape 46 : Serrer sans forcer pour éviter de déformer le rail.
  • Étape 47 : Installer la stabilisation centrale.
  • Étape 48 : Vérifier la vis de la stabilisation latérale.
  • Étape 49 : Installer les stabilisations latérales
  • Étape 50 : Tester l’équilibre général de l’arc en main.
  • Étape 51 : Sortir les flèches et les vérifier une par une.
  • Étape 52 : Regarder si les tubes sont droits.
  • Étape 53 : Vérifier que les plumes sont bien collées.
  • Étape 54 : Vérifier que les pointes sont bien fixées.
  • Étape 55 : Vérifier que les encoches ne sont pas fissurées.
  • Étape 56 : Identifier la plume coq (plume de couleur différente).
  • Étape 57 : Orienter la flèche plume coq vers l’extérieur du repose-flèche.
  • Étape 58 : Vérifier le clipsage de l’encoche sur la corde.
  • Étape 59 : Ajuster l’encoche si elle serre trop ou pas assez.
  • Étape 60 : Vérifier le spine des flèches.
  • Étape 61 : Contrôler la longueur totale des flèches.
  • Étape 62 : S’assurer que rien ne frotte contre la plume lors du tir.
  • Étape 63 : Mettre le protège-bras.
  • Étape 64 : Ajuster la lanière pour qu’il ne tourne pas.
  • Étape 65 : Mettre la palette.
  • Étape 66 : Mettre la dragonne de la poignée.
  • Étape 67 : Se tenir sur le pas de tir correctement.
  • Étape 68 : Écarter les pieds à largeur d’épaules.
  • Étape 69 : Relâcher épaules et nuque.
  • Étape 70 : Analyser la lumière et les conditions météo.
  • Étape 71 : Vérifier que rien n’obstrue le champ de tir.
  • Étape 72 : Regarder la distance de tir et ajuster le viseur.
  • Étape 73 : Vérifier les vis du viseur une dernière fois (comme ça il tombe pas).
  • Étape 74 : Placer la flèche correctement sur le repose-flèche.
  • Étape 75 : Clipper l’encoche sur la corde.
  • Étape 76 : Vérifier que la flèche est bien orientée plume coq vers l’extérieur.
  • Étape 77 : Vérifier que personne n’est devant la ligne de tir.
  • Étape 78 : Monter l’arc à hauteur de la cible.
  • Étape 79 : Amener la main de corde vers le point d'ancraged ancrage.
  • Étape 80 : Prendre conscience de l’alignement corps–arc–cible.
  • Étape 81 : Tirer la corde par la force du dos, pas des bras.
  • Étape 82 : Atteindre ton point d’ancrage habituel.
  • Étape 83 : Stabiliser le viseur
  • Étape 84 : Relâcher la corde proprement, sans écarter les doigts vers l’extérieur.
  • Étape 85 : Maintenir la posture après le tir
  • Étape 86 : Observer le vol de la flèche et son impact.
  • Étape 87 : Analyser le résultat.
  • Étape 88 : RECOMMENCER
